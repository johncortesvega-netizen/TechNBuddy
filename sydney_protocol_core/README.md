# Sydney Protocol Core Package

This package contains the importable framework code for Sydney Protocol Core v0.3.

Use `analyze_text()` to produce bounded readings and `create_receipt()` to produce compact trace receipts. The core is suitable for apps such as TechnBuddy, Speech Reader, AI Claim Reader, or Receipt Reader, but it does not contain those apps.
